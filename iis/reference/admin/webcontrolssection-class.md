# WebControlsSection Class
Specifies the shared location of client script files.  
  
## Syntax  
  
```vbs  
class WebControlsSection : ConfigurationSection  
```

## Properties  
 The following table lists the properties exposed by the `WebControlsSection` class.  
  
|Name|Description|  
|----------|-----------------|  
|`ClientScriptsLocation`|A required read/write `string` value that specifies the location for client scripts. The default is "/aspnet_client/{0}/{1}/." **Note:**  The "{0}" will be replaced with the namespace name (for example, system_web), and the "{1}" will be replaced with the [!INCLUDE[dnprdnshort](../../reference/admin/includes/dnprdnshort-md.md)] version number.|  
|`Location`|(Inherited from `ConfigurationSection`.) A key property.|  
|`Path`|(Inherited from `ConfigurationSection`.) A key property.|  
|`SectionInformation`|(Inherited from `ConfigurationSection`.)|  
  
## Subclasses

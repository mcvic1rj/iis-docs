# IsapiFiltersSection Class
Configures the ISAPI filters on a Web server.  
  
## Syntax  
  
```vbs  
class IsapiFiltersSection : ConfigurationSectionWithCollection  
```

## Properties  
 The following table lists the properties exposed by the `IsapiFiltersSection` class.  
  
|Name|Description|  
|----------|-----------------|  
|`IsapiFilters`|An array of read/write [IsapiFilterElement](../../reference/admin/isapifilterelement-class.md) values that represent the ISAPI filters on a Web server.|  
|`Location`|(Inherited from `ConfigurationSection`.) A key property.|  
|`Path`|(Inherited from `ConfigurationSection`.) A key property.|  
|`SectionInformation`|(Inherited from `ConfigurationSection`.)|  
  
## Subclasses  
 This class contains no subclasses.
